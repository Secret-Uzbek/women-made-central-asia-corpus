# Raw Upload Inbox

This folder is the upload inbox for raw source materials of the Women Made Central Asia Corpus.

Upload here first:

- PDF files;
- DOCX files;
- scanned books and articles;
- ZIP archives;
- source bundles;
- exported Google Docs.

After upload, every file must receive:

1. a stable source ID;
2. a source card;
3. a reading status;
4. an extraction or OCR status;
5. a rights/publication status;
6. a deduplication status;
7. a next corpus action.

Do not treat Google Docs `.gdoc` pointer files as raw documents. Export them as DOCX or PDF first.
